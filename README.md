# Vanity Salon Application

## Overview
The Vanity Salon application is designed to provide a seamless experience for users to check in or sign up for membership at the salon. The application features a user-friendly interface that guides users through the process of entering their information, selecting a membership plan, and confirming their subscription.

## Project Structure
```
vanity-salon
├── index.html        # Main HTML structure for the application
├── css
│   └── styles.css    # Styles for the application
├── js
│   └── app.js        # JavaScript functionality for the application
├── .gitignore        # Git ignore file
└── README.md         # Documentation for the project
```

## Setup Instructions
1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd vanity-salon
   ```

2. **Open the Application**
   - Open `index.html` in your web browser to view the application.

## Features
- **User Navigation**: Users can navigate through different pages including welcome, information form, membership selection, and thank you page.
- **Form Handling**: Users can input their name and contact information, which is validated before proceeding.
- **Membership Selection**: Users can choose from various membership options with clear pricing.
- **Responsive Design**: The application is designed to be visually appealing and functional on various screen sizes.
